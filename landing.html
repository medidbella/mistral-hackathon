Installation Guide & Project Overview

Problem

Social media addiction is a real challenge for productivity. We often find ourselves distracted by platforms like Instagram, Twitter, and YouTube when we should be focused on work. The AI Bouncer extension aims to tackle this problem by adding friction to the process of accessing distracting websites.

The Idea

The AI Bouncer works as a browser extension that intercepts attempts to access social media sites. Users must provide a reason for needing to visit the site. This reason is evaluated by an AI (powered by Mistral) which checks previous approvals and daily social media usage. Based on this, the AI either grants a timed pass, roasts the user, provides advice, or blocks the page.

Project Architecture

Backend (Fastify + SQLite)

  • Fastify: Handles requests and AI evaluation.
  • SQLite: Stores user interactions and social media usage data.
  • Mistral AI: Evaluates reasons and provides a pass or advice.

Browser Extension

  • Intercepts page loads for sites like Instagram, Twitter, YouTube.
  • Shows a popup asking for a reason if the site is blocked.
  • AI evaluates the reason and decides whether to grant access.

How to Run the Project Locally

1. Set Up the Backend

cd backend
npm install
cp env.sample .env
# Add your Mistral API key
MISTRAL_API_KEY=your_key_here
npm run build
npm start
# Backend runs at http://localhost:3000

2. Install the Extension

  1. Open Google Chrome or any Chromium-based browser.
  2. Go to chrome://extensions/ and enable Developer mode.
  3. Click "Load unpacked" and select the extension folder.

3. Test the AI Bouncer

  1. Try visiting a distracting site (e.g.,) https://youtube.com
  2. The AI Bouncer will block the page and ask for a reason.
  3. Submit your excuse and see if Mistral AI grants access.
